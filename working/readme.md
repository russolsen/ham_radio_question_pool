= Releasing a new test

To release a new test:

* Make a directory with the appropriate name here.

* Copy the pdf file into the directory.

* Copy one of the Makefiles into the directory.

* Edit the Makefile.

* Add the directory to the working/Makefile.

* Do a `make build` in the directory to generate the files.

* Do a `make check` in the directory to do the basic checks.

* Spot check the result.

* Do a `make install` in the directory.

* Check the resulting release directory in.

* Check in the original file and the Makefile.

* Celebrate.
